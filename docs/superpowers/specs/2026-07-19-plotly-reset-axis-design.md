# Plotly 축 원복 버튼 설계

## 목표

생활정보 대시보드의 모든 Plotly 그래프에서 확대 후 초기 X·Y축 범위로 돌아갈 수 있는 원복 버튼 1개를 제공한다.

## UI 동작

- 그래프에 마우스를 올리면 오른쪽 위에 Plotly 기본 modebar가 반투명하게 나타난다.
- modebar에는 `resetScale2d`(축 원복) 버튼만 표시한다.
- 버튼을 누르면 확대·이동된 X축과 Y축이 그래프 생성 시 지정된 초기 범위로 함께 돌아간다.
- 다운로드, 확대, 축소, 자동축, 선택, 라쏘 버튼은 표시하지 않는다.

## 구현 구조

- `utils/charts.py`에 공용 Plotly 설정 상수를 1개 둔다.
- 공용 차트 렌더링 호출 4곳이 같은 설정을 사용한다.
- `views/00_Home.py`의 홈 미니 그래프 1곳도 같은 설정을 사용한다.
- 데이터 조회, 차트 데이터, Y축 초기 범위 계산은 변경하지 않는다.

## 검증

- 정적 검사: Plotly 호출부 5곳 모두 공용 설정 사용, `resetScale2d` 포함, 불필요 버튼 제외.
- 구문 검사: 수정 파일 2개의 `py_compile` 오류 0건.
- AppTest: 홈과 날씨 페이지 예외 0건, 날씨 Plotly 그래프 2개 유지.
- 독립 체커: 수용기준 대비 `PASS` 또는 `NEEDS_WORK` 판정.
